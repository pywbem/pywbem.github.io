Module Hierarchy

  • pywbem: The pywbem package provides a WBEM client library and some WBEM utility commands, all written in pure Python.
    • pywbem.cim_constants: Useful CIM constants.
    • pywbem.cim_http: Send HTTP/HTTPS requests to a WBEM server.
    • pywbem.cim_obj: Representations of CIM objects, and a case-insensitive dictionary.
    • pywbem.cim_operations: CIM operations over HTTP.
    • pywbem.cim_types: Types to represent CIM typed values, and related conversion functions.
    • pywbem.cim_xml: Functions to create XML documens and elements conforming to the DMTF standard DSP0201, Representation of CIM in XML, v2.2.
    • pywbem.cimxml_parse: CIM/XML Parser.
    • pywbem.mof_compiler: pywbem MOF Compiler Rules and implementation definition for lex and yacc.
    • pywbem.tupleparse: Tuple parser for the XML schema representing CIM messages.
    • pywbem.tupletree: tupletree - Convert XML DOM objects to and from tuple trees.
    • pywbem.wbemcli: A small utility to wrap up a PyWBEM session in a Python interactive console.
