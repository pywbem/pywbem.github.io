 22  """ 
 23  tupletree - Convert XML DOM objects to and from tuple trees. 
 24   
 25  DOM is the standard in-memory representation of XML documents, but it 
 26  is very cumbersome for some types of processing where XML encodes 
 27  object structures rather than text documents.  Direct mapping to Python 
 28  classes may not be a good match either. 
 29   
 30  tupletrees may be created from an in-memory DOM using 
 31  dom_to_tupletree(), or from a string using xml_to_tupletree(). 
 32   
 33  Since the Python XML libraries deal mostly with Unicode strings they 
 34  are also returned here.  If plain Strings are passed in they will be 
 35  converted by xmldom. 
 36   
 37  Each node of the tuple tree is a Python 4-tuple, corresponding to an 
 38  XML Element (i.e. <tag>): 
 39   
 40    (NAME, ATTRS, CONTENTS, None) 
 41   
 42  The NAME is the name of the element. 
 43   
 44  The ATTRS are a name-value hash of element attributes. 
 45   
 46  The CONTENTS is a list of child elements. 
 47   
 48  The fourth element is reserved. 
 49  """

 64      if node.nodeType == node.DOCUMENT_NODE:

 73      for child in node.childNodes:

 77              assert isinstance(child.nodeValue, six.string_types), \ 
 78                     "text node %s is not a string %r" % child

 81              contents.append(child.nodeValue) 
 82          else: 
 83              raise RuntimeError("can't handle %r" % child)
