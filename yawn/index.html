YAWN

Overview

YAWN (repository) stands for "Yet Another WBEM Navigator" and provides a way to access WBEM servers and to navigate between the CIM objects returned.

YAWN runs in the Apache web server, it is Python based and utilizes Apache's `mod-python`.

This project is experimental.
